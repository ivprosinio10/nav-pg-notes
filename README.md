# nav-pg-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容主要涉及各类导航、笔记或信息整理，不针对任何特定域名或网站。本仓库旨在以静态 HTML 的形式保存和展示这些页面，方便查阅与分享。

## 目录说明

```
/
├── index.html          # 仓库主页，列出所有归档页面链接
├── pages/              # 存放所有独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 静态资源文件（如图片、样式、脚本）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库入口页面，提供导航列表。
- `pages/`：所有归档的 HTML 页面均放置于此目录下。
- `assets/`：存放页面所需的公共资源，按类型分类。

## 页面归档说明

- 每个独立页面均为自包含的 HTML 文件，可直接在浏览器中打开查看。
- 页面内容以信息整理和导航为主，不包含商业推广或特定营销内容。
- 归档的页面可能不定期更新或新增，历史版本可通过 Git 提交记录追溯。

## 维护说明

- 本仓库由维护者手动更新，欢迎通过 Issue 或 Pull Request 提交建议或修正。
- 提交新页面时，请将其放入 `pages/` 目录，并在 `index.html` 中添加对应链接。
- 请勿在页面中添加任何形式的跟踪代码、广告或恶意脚本。
- 保持页面简洁、清晰，避免引入过多外部依赖。
